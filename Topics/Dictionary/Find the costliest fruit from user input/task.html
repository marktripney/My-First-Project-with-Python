Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that reads a dictionary of fruits and their prices from the user. The user will input the fruit name and price as key-value pairs, with each pair on a separate line. The input will end with a blank line. Your program should then find and print the name of the fruit with the highest price. If there are ties, print any of the fruits with the highest price.
Sample Input:
apple
1.5
banana
2.0
orange
1.75


Sample Output:
banana


Sample Input:
pear
2.5
grape
3.0
watermelon
4.0


Sample Output:
watermelon



Memory limit: 256 MB
Time limit: 15 seconds
